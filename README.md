# Python List Playground

A beginner-friendly repository to practice and explore **Python list operations**.  
This project demonstrates how to use common list methods such as `copy()`, `insert()`, `remove()`, `pop()`, `index()`, `sort()`, and `clear()`.

---

## 🚀 Features
- Copying lists
- Inserting elements at specific positions
- Removing and popping elements
- Finding indexes and checking membership
- Sorting lists (ascending & descending)
- Clearing all elements

---

## 📂 Project Structure
- `list_operations.py` → Contains examples of different list methods
